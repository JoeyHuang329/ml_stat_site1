SECTION 01

研究問題形式化

在動手寫程式之前，先把問題講清楚是良好研究的第一步。令七維心理計量特徵空間為

$$\mathcal{X}_{\text{psych}} = \{\text{Nscore, Escore, Oscore, Ascore, Cscore, Impulsive, SS}\}$$

它由三套量表組成：NEO-FFI-R 五大人格（神經質、外向、開放、友善、嚴謹）、BIS-11 衝動性，以及 ImpSS 感覺尋求。對每一種藥物 $d$（共 $|\mathcal{D}|=15$ 種），我們想知道是否存在一個預測函數

$$f_d : \mathbb{R}^7 \rightarrow [0,1]$$

能把一個人的七維人格分數映射到「該藥物高風險使用」的機率，而且這個預測明顯優於隨機猜測。把「明顯優於隨機」寫成統計假設，就是對每種藥物建立一組單側檢定：

$$H_0^{(d)}: \text{AUC}_d = 0.5 \quad \text{vs.} \quad H_1^{(d)}: \text{AUC}_d > 0.5$$
觀念釐清：為什麼「AUC = 0.5」代表隨機猜測？

AUC（ROC 曲線下面積）有一個很直觀的機率定義：隨機抽一個真正的使用者與一個非使用者，模型給使用者較高分數的機率，即 $\text{AUC}=\mathbb{P}\big(\hat{p}(\mathbf{x}\mid y=1) > \hat{p}(\mathbf{x}\mid y=0)\big)$。如果模型毫無鑑別力，這個機率就是丟硬幣的 $0.5$；完美排序則是 $1.0$。所以「檢定 AUC 是否大於 0.5」等同於問「人格分數到底有沒有把使用者排在非使用者前面的能力」。它的好處是不受分類門檻與正負樣本比例影響，特別適合本研究中正樣本比例從 4% 到 56% 差異巨大的情境。

1.1　目標變數的兩種操作化

原始資料把每種藥物的使用頻率記為 CL0 到 CL6 七個有序等級（從「從未使用」到「昨天使用過」）。本研究依不同模型需要，把它操作化為兩種形式。

(a) 二值化高風險指標（供 Lasso、KNN）

$$y_i^{(d)} = \mathbf{1}\!\left[\text{CL}_i^{(d)} \in \{\text{CL3, CL4, CL5, CL6}\}\right]$$

以 CL3（過去一年內使用）為閾值，把「近期活躍使用者」標為 1、其餘標為 0。這個切點在心理測量上能最有效區分「活躍使用者」與「歷史接觸者」，並讓十五種藥物的正樣本比例落在 4.26%（crack）到 56.12%（nicotine）之間，避免極端不均衡使 AUC 退化。

(b) 序列等距化（供 Ridge 基準）

$$z_i^{(d)} = k, \quad \text{若 } \text{CL}_i^{(d)} = \text{CL}k, \quad k \in \{0, 1, \ldots, 6\}$$

直接把 CL0–CL6 當成 0–6 的連續分數。這隱含一個強假設：相鄰等級之間的「心理距離」相等（CL0→CL1 與 CL5→CL6 一樣遠）。這假設並不真實，因此 Ridge 只當作參照基準，不作為主要推論依據。

1.2　排除規則

最終的十五種藥物排除了兩類物質，各有統計理由：

  • 虛構藥物 Semeron（Semer）：這是問卷裡的「陷阱題」——一種現實中不存在的藥物。宣稱使用過它的人就是不實填答者，用來偵測並剔除亂答者。
  • 合法日常物質 Alcohol、Caffeine、Chocolate：它們的正樣本比例都超過 90%，幾乎人人使用，使得 AUC 退化、無法形成有意義的預測問題。

剩下的十五種建模目標為：amphet、amyl、benzos、cannabis、coke、crack、ecstasy、heroin、ketamine、legalh、lsd、meth、mushrooms、nicotine、vsa。

2.1　資料清洗與類別特徵編碼

第一步是剔除 8 位宣稱使用過虛構藥物 Semeron 的不實填答者，樣本從 $N=1{,}885$ 降為 $N=1{,}877$。若保留這些「錯誤訊號」，它們會以離群值的形式拉扯正規化路徑，產生虛假的非零係數。

類別特徵的傾斜很嚴重：族裔中 White 佔 91.25%，居住國中 UK 佔 55.38%、USA 佔 29.55%。直接做 One-Hot 編碼會出問題——以佔比僅 0.16% 的稀少類別為例，其虛擬變數變異數趨近零，在 $\ell_1$ 懲罰下幾乎必然被「隨機捨棄」，造成虛假的特徵篩選。因此：

  • 族裔二值化為 $\mathbf{1}[\text{White}]$，把 6 個自由度壓成 1 個。
  • 居住國分為 UK / USA / Other 三類後做 Target Encoding（以 $\widehat{\mathbb{E}}[y\mid\text{Country}]$，即各國的高風險比例，取代類別標籤），用單一連續維度承載組間資訊。
  • 七維心理特徵做 StandardScaler 標準化，使係數可以跨特質直接比較大小。

2.2　共線性：感覺尋求與衝動性的糾纏

在七維人格特徵兩兩的 Spearman 等級相關中，最強的一對是 SS（感覺尋求）↔ Impulsive（衝動性），$\rho = 0.63$。兩者共享一個高層次構念「抑制解除（Disinhibition）」。其餘較顯著的還有 Oscore–SS（+0.41）、Nscore–Escore（−0.42）、Nscore–Cscore（−0.38）。

值得注意的是，變異數膨脹因子（VIF）診斷顯示所有特徵的 VIF 都低於 1.92，遠在警戒線 5 以下——這代表沒有「嚴重」共線性會讓設計矩陣接近奇異。但 $\rho=0.63$ 仍足以威脅 Lasso 的特徵選擇穩定性。

觀念釐清：VIF 不高，為什麼還要擔心共線性？

這是兩個不同尺度的指標。$\rho=0.63$ 衡量的是兩個特徵之間的雙變量依賴（點對點）；VIF 衡量的是所有其他特徵聯合對某特徵的解釋力（多對一的全局指標）。要讓 OLS 設計矩陣接近奇異、VIF 飆高，通常需要 $|\rho|>0.9$ 級別的相關，所以這裡 VIF 才會偏低。然而對 Lasso 而言，問題不在矩陣奇異，而在於：當兩個特徵高度相關時，$\ell_1$ 懲罰傾向「擇一保留、丟棄另一個」，而保留哪一個在不同訓練集上可能隨機翻轉，導致特徵選擇結果不穩定。後文會看到，這正是 SS 與 Impulsive 在 Lasso 中表現的對比訊號。

2.3　高風險族群的「風險三角」

2.4　國籍：一個必須處理的干擾因子

各國的高風險使用率差異巨大：USA（39.75%）約是 UK（12.84%）的三倍。卡方獨立性檢定得到 $\chi^2 = 160.53$、$p = 1.39\times10^{-35}$，國籍與高風險使用顯著相關。這是一個極強的干擾變數（Confounder）。

觀念釐清：干擾因子、捷徑學習，與辛普森悖論

如果不處理國籍，模型可能學到一條捷徑：「特徵看起來像美國人 → 預測高風險」，完全繞過人格這條真實路徑，違背研究初衷。處理方式是把 Country 設為非懲罰性共變數，讓正規化懲罰只作用在人格特徵上，估出的人格係數因此代表「在相同國籍條件下」的邊際效應。

更進一步，研究還檢查了辛普森悖論：會不會「整體看來 SS 與用藥正相關，但分國籍後反而變負相關」？分層計算 SS 與古柯鹼使用的 Spearman 相關，三組（UK 0.340、USA 0.161、Other 0.215）都是正向且顯著，方向一致——悖論不存在，SS 可以安心作為核心預測因子。

觀念釐清：決定性編碼 vs 資料相依編碼，為何只有後者怕洩漏？

國籍的三分組（把實數對應到 UK/USA/Other 字串）是決定性的，可以安全地在切分前一次做完；但隨後的 Target Encoding（把國籍換成該組的高風險比例 $\widehat{\mathbb{E}}[y\mid\text{Country}]$）需要用到 $y$，是資料相依的，必須留在「只在訓練折擬合」的位置。程式因此刻意把兩步拆開：CountryGrouper 只做字串映射，真正用到標籤的 TargetEncoder 交給下一節的 ColumnTransformer 在折內處理。這個拆分正是「哪些步驟可以提前、哪些必須折內」的判斷準則。

3.1　前處理模組：兩個變體

前處理用 scikit-learn 的 ColumnTransformer 封裝。它有兩個工廠函數，因為不同模型需要不同的特徵空間。

觀念釐清：資料洩漏與「折內擬合」

StandardScaler 要算平均與標準差 $(\hat{\mu},\hat{\sigma}^2)$，TargetEncoder 要算 $\widehat{\mathbb{E}}[y\mid\text{Country}]$——這些都是資料相依的量。如果先用「全部資料」算好再切訓練/驗證，驗證集的資訊就會偷偷流入訓練過程，這叫資料洩漏（Data Leakage），會讓效能看起來虛假地好（樂觀偏誤）。

正確做法是「只在訓練折上 fit，再 transform 驗證折」。把這些步驟封裝進 Pipeline 後，當外層做交叉驗證時 scikit-learn 會自動在每一折內重新擬合，從根本上隔絕洩漏。這就是把前處理寫成 Pipeline 而非手動先標準化的關鍵理由。

3.2　分層切分稽核：可比性的地基

在訓練任何模型前，流水線對十五種藥物各自做一次 80:20 的分層訓練/測試切分（分層變數是該藥物的二值高風險標籤），並稽核訓練集與測試集的正樣本比例是否一致。

4.1　Ridge　未稀疏化的對照地圖

Ridge 在連續化等級 $z\in\{0,\ldots,6\}$ 上做 $\ell_2$ 懲罰的線性迴歸：

$$\hat{\boldsymbol{\beta}}^{\text{Ridge}}_d = \arg\min_{\boldsymbol{\beta}} \left\{ \frac{1}{2n} \sum_{i=1}^{n} \left(z_i^{(d)} - \beta_0 - \mathbf{x}_i^{\top} \boldsymbol{\beta}\right)^2 + \lambda \|\boldsymbol{\beta}\|_2^2 \right\}$$

$\ell_2$ 懲罰不會把任何係數壓成零，而是在共線特徵之間「均攤」效應（shrinkage）。例如 SS 與 Impulsive 高度相關時，Ridge 會把訊號分給兩者。它因此提供一張「保留全部訊號」的線性邊際效應地圖。超參數 $\lambda$ 在 $10^{-4}$ 到 $10^{2}$ 的 13 點對數網格上，用 10-Fold CV 以負 MSE 選出。

4.2　Lasso　主模型與對數勝算比

Lasso 在二值目標上做 $\ell_1$ 懲罰的邏輯迴歸：

$$\hat{\boldsymbol{\beta}}^{\text{Lasso}}_d = \arg\min_{\boldsymbol{\beta}} \left\{ -\frac{1}{n} \sum_{i=1}^{n} \left[y_i \log p_i + (1 - y_i)\log(1 - p_i)\right] + \lambda \|\boldsymbol{\beta}\|_1 \right\}$$

其中 $p_i = \sigma(\beta_0 + \mathbf{x}_i^{\top}\boldsymbol{\beta})$、$\sigma(z)=1/(1+e^{-z})$ 為 sigmoid 函數。因為特徵已標準化，係數有直接的詮釋：

$$\exp(\hat{\beta}_j) = \text{勝算比（Odds Ratio）}：\text{該特質提升 1 個標準差，使用 odds 變為原來的幾倍}$$

對極不均衡的藥物（訓練集正樣本比例 $<15\%$，含 crack、heroin、vsa、amyl、ketamine）啟用 class_weight='balanced'，等價於給少數類樣本反頻率加權。超參數 $C=1/\lambda$ 在 $10^{-3}$ 到 $10^{2}$ 的 11 點對數網格上，以 AUC 選出。

觀念釐清：$\ell_1$ 與 $\ell_2$ 的幾何差別——為什麼 Lasso 會「歸零」？

正規化是在「最小化誤差」與「限制係數大小」之間取捨。$\ell_2$ 把係數限制在一個圓形區域內，$\ell_1$ 則限制在一個菱形（有尖角）區域內。誤差等高線通常會先碰到菱形的尖角，而尖角正好落在某些係數等於零的座標軸上——這就是 $\ell_1$ 會把不重要的係數直接壓成 0（自動做特徵選擇）、而 $\ell_2$ 只會把係數縮小卻不歸零的幾何直覺。

實務後果：Ridge 的地圖密密麻麻全是非零值，適合「看全貌」；Lasso 的地圖稀疏、只剩存活的特徵，適合「挑出真正重要的訊號」。而「哪些被歸零」本身就是診斷資訊——如後文所示，Impulsive 屢被剔除、SS 從未被剔除。

Bootstrap 信賴區間

我們想知道每個 Lasso 係數是否「顯著不為零」，但 Lasso 的解沒有像普通迴歸那樣的封閉式常態分布（它在 $\beta_j=0$ 處有一塊「點質量」，不是平滑的高斯），不能直接套用一般的檢定公式。解法是非參數百分位 Bootstrap：

  1. 從訓練集放回抽樣 $n$ 筆，組成一份新樣本；
  2. 固定 $\lambda=\hat{\lambda}^{\text{CV}}$ 重新擬合，得到一組係數 $\hat{\boldsymbol{\beta}}^{*(b)}$；
  3. 重複 $B=1{,}000$ 次，得到每個係數的 1000 個估計值；
  4. 取第 2.5 與 97.5 百分位作為 95% 信賴區間 $[Q_{2.5}, Q_{97.5}]$。

若這個區間不包含 0，就說該係數在 95% 信賴水準下顯著。

觀念釐清：Bootstrap 在做什麼？

我們手上只有一份訓練資料，無法真的「再抽很多份樣本」來看估計值的波動。Bootstrap 的巧思是：把手上的樣本當成母體，反覆放回抽樣模擬「重新蒐集資料」。每次抽樣得到的係數都略有不同，這 1000 個係數的分散程度，就近似了「如果重新做一次研究，係數會怎麼變動」。當這些係數有夠大比例落在零的同一側（區間不跨零），我們才有信心說效應是真的，而不是抽樣運氣。

4.3　KNN　線性是否足夠的試金石

KNN 對一個新樣本 $\mathbf{x}_0$，找出它在七維標準化心理空間中歐式距離最近的 $k$ 個鄰居，用這些鄰居的標籤平均作為機率預測：

$$\hat{p}^{\text{KNN}}_d(\mathbf{x}_0) = \frac{1}{k} \sum_{i \in \mathcal{N}_k(\mathbf{x}_0)} y_i^{(d)}$$

KNN 是純粹的無母數方法，理論上能逼近任意形狀的決策邊界。它在這裡扮演診斷角色：如果 KNN 的 AUC 遠勝 Lasso，代表資料中存在線性模型抓不到的局部群集或強交互作用，也就是線性附加假設「結構誤設」；反之，若 Lasso 不輸 KNN，就支持「線性已足夠」。超參數 $k \in \{5,7,10,15,25,50,100\}$，以 AUC 選出。

為什麼 KNN 只用七維心理特徵？ 歐式距離要在同質的連續空間才有意義；維度 $p=7$ 相對於樣本量 $n\approx1500$ 足夠低，維度詛咒影響輕微。這也讓 Lasso 與 KNN 的比較聚焦在「同一組人格訊號上，線性 vs 無母數」的純粹對照。

5.2　三個推論程序

(1) AUC > 0.5 單側檢定（Hanley & McNeil, 1982）

用 AUC 的漸近常態變異公式估出標準誤，再算檢定統計量 $Z = (\widehat{\text{AUC}} - 0.5)/\widehat{\text{SE}}$，回答「人格對該藥物是否真有預測力」。

$$\widehat{\text{Var}}(\widehat{\text{AUC}}) = \frac{\widehat{\text{AUC}}(1-\widehat{\text{AUC}}) + (m-1)(Q_1 - \widehat{\text{AUC}}^2) + (n-1)(Q_2 - \widehat{\text{AUC}}^2)}{mn}$$

(2) DeLong 配對檢定（DeLong et al., 1988）

在同一份測試集上比較 Lasso 與 KNN 的 AUC 是否有顯著差異。它利用 AUC 的 U 統計量分解與其結構分量的共變數構造檢定量：

$$Z_{\text{DeLong}} = \frac{\widehat{\text{AUC}}_{\text{Lasso}} - \widehat{\text{AUC}}_{\text{KNN}}}{\sqrt{\frac{1}{m} S_{10}^{(1,2)} + \frac{1}{n} S_{01}^{(1,2)}}}$$

正的 $z$ 代表 Lasso 較佳。這是判斷「線性是否結構誤設」的核心工具。

(3) Benjamini–Hochberg FDR 校正（1995）

因為我們對 15 種藥物各做一次檢定，需要校正多重比較。

觀念釐清：為什麼要做多重比較校正，以及 FDR 與傳統校正的差別

做一次檢定、設 $\alpha=0.05$，代表「就算沒效，也有 5% 機率誤判為有效」。但做 15 次檢定，至少誤判一次的機率會大幅上升。傳統的 Bonferroni 把門檻壓到 $0.05/15$，非常保守，容易把真效應也一起殺掉。

Benjamini–Hochberg 控制的是偽發現率（FDR）——「在所有被宣告為顯著的結果中，預期有多少比例其實是假的」。它把 $p$ 值由小到大排序後給予遞增的門檻，比 Bonferroni 寬鬆、檢定力更高，是現代多重檢定的常用選擇。校正後的值稱為 $q$ 值；$q<0.05$ 才算顯著。

可重現性：統一隨機種子 GLOBAL_SEED = 42，Bootstrap 內外層獨立播種（內層 $\text{seed}+b+1$）避免狀態交叉污染；交叉驗證為 10-Fold Stratified；Bootstrap $B=1{,}000$ 次。
觀念釐清：AUC-ROC 其實就是 Mann–Whitney U

§1 提到 AUC 等於「隨機抽一正一負、模型把正樣本排前面的機率」。這個量正好等於無母數的 Mann–Whitney U 統計量除以 $mn$（$m$、$n$ 為正負樣本數）。也就是說，AUC 本質上是在問「兩群分數的秩次分離得多開」，完全不依賴分數的絕對尺度或分布形狀。這就是為什麼在類別不均衡、機率未校準的情況下，AUC 仍是穩健可比的排序型指標。

Benjamini–Hochberg FDR 校正

把一組 $p$ 值由小到大排序，乘上遞增因子 $n/k$，再由後向前取累積最小值以保證單調，最後裁切到 $[0,1]$ 得到 $q$ 值。短短幾行，就是現代多重檢定最常用的校正。

evaluation/inference.py　·　benjamini_hochbergpython
    """
    p = np.asarray(pvals, dtype=float).copy()
    p[np.isnan(p)] = 1.0
    n = len(p)
    order = np.argsort(p)
    ranked = p[order]
    # adjusted p_(i) = min_{k >= i} ( n / k · p_(k) )，由大到小回掃保證單調
    factor = n / np.arange(1, n + 1)
    raw_adj = ranked * factor
    # 由後向前取累積最小值，再回填位置
    adj_sorted = np.minimum.accumulate(raw_adj[::-1])[::-1]
    adj_sorted = np.clip(adj_sorted, 0.0, 1.0)
    pvals_corrected = np.empty(n, dtype=float)
    pvals_corrected[order] = adj_sorted
    reject = pvals_corrected <= alpha
    return reject, pvals_corrected

主要模式可整理為：

  • SS（感覺尋求）是普適正向因子：十五種藥物全為正，範圍約 $[0.06, 0.35]$，無任何正負反轉。

  • Nscore（神經質）對成癮/鎮靜型正向：最突出是 benzos（+0.42）。

6.4　結構誤設診斷：線性夠不夠？

這是全研究最關鍵的方法論判斷。把每種藥物畫成一個點，橫軸是 Lasso AUC、縱軸是 KNN AUC，再畫一條 $y=x$ 對角線。

雖然 Lasso 的 AUC 點估計普遍高於 KNN（最大差距如 meth +0.138、lsd +0.131），但 DeLong 配對檢定的原始 $p$ 值範圍是 $[0.029, 0.927]$，經 BH-FDR 校正後全部 $q$ 值都大於 0.05。例如差距最大的 cannabis，DeLong $z=2.18$、$p=0.029$，但校正後 $q=0.433$，不顯著。

再加上一個事實：KNN 選出的最佳鄰居數 $k$ 普遍偏大（多為 100，少數 50），意味它偏好大鄰域的全域平滑，而非依賴局部群集。把這兩點合起來，得到本研究最重要的推論之一：

線性附加模型族沒有結構誤設

七維人格與藥物使用的關聯，主要是平滑、近似線性附加的全域結構，而非需要無母數方法才能捕捉的局部群集或強交互作用。因此（一）無需引入樹模型或深度網路，Lasso 已足夠；（二）報告中的 log-odds 係數可直接作為心理測量學詮釋的基礎，不必擔心「真實的非線性訊號被線性模型遺漏」。

6.5　Ridge 與 Lasso 的相互印證

兩個正交的正規化框架（$\ell_2$ 均攤 vs $\ell_1$ 稀疏）給出方向完全一致的結論：SS 普適正向、Oscore 對迷幻藥群正向、Cscore 對 cannabis/nicotine 負向，無任何方向反轉。差別僅在 Lasso 把訊號集中、絕對值較大（如 SS-cannabis 在 Lasso 為 0.53、Ridge 為 0.34，但兩者尺度不同，不可直接比大小）。方向一致性說明這些結論是穩健的資料特性，而非單一模型的工件（artifact）。
